# recipeAPI
 recipe api
step 1 : npm i
step 2 : node app.js
CRUD API

    GET /findall
    get all recipes from database
    POST /save/
    save a recipe to the database
    PUT /update
    update a recipe in the database
    DELETE /delete/
    delete a recipe from the database
